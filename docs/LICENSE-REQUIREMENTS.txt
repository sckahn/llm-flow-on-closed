1. 유료 라이선스 필요 (상용 환경)
================================================================================

[Docker Desktop]
--------------------------------------------------------------------------------
- 조건: 직원 250명 이상 또는 연매출 $10M 이상 기업은 유료
- 가격: $5~$24/user/month (팀 규모에 따라)
- 대안:
  * Podman (무료, Red Hat)
  * containerd + nerdctl (무료)
  * Rancher Desktop (무료)
  * Docker Engine (Linux 서버에서 무료, CLI만 사용)
- 권장: Linux 서버에서 Docker Engine 사용 시 무료

[Neo4j Enterprise Edition]
--------------------------------------------------------------------------------
- 조건: Community Edition은 GPL v3로 무료, 하지만 다음 기능은 Enterprise 필요:
  * Clustering / HA
  * Role-based access control
  * Hot backup
  * 성능 모니터링 고급 기능
- 가격: 문의 필요 (일반적으로 연간 수만~수십만 달러)
- 대안:
  * Community Edition 사용 (단일 노드, 기본 기능)
  * Amazon Neptune (AWS 종량제)
  * ArangoDB (Apache 2.0)
- 현재 설정: Community Edition 사용 중 (무료)

[Grafana Enterprise]
--------------------------------------------------------------------------------
- 조건: OSS 버전은 AGPLv3로 무료, Enterprise 기능 사용 시 유료:
  * LDAP/SAML 고급 인증
  * 엔터프라이즈 플러그인
  * 리포팅
- 가격: 문의 필요
- 현재 설정: OSS 버전 사용 중 (무료)

[MinIO Enterprise]
--------------------------------------------------------------------------------
- 조건: AGPLv3 오픈소스는 무료, 상용 지원 필요 시 유료
  * Standard: $10/TB/month
  * Enterprise: 문의 필요
- 현재 설정: AGPLv3 버전 사용 중 (무료, 소스 공개 의무 있음)

4. AGPLv3 주의사항
================================================================================

다음 컴포넌트는 AGPLv3 라이선스입니다:
- Grafana OSS
- MinIO
- Neo4j Community Edition (GPLv3)

AGPLv3/GPLv3 의무사항:
1. 소스 코드 공개: 수정한 경우 소스 공개 필요
2. 네트워크 사용: 네트워크로 서비스 제공 시에도 소스 공개 의무
3. 동일 라이선스: 파생 저작물도 동일 라이선스 적용

권장 조치:
- 해당 컴포넌트를 수정하지 않고 그대로 사용
- 별도 서비스로 분리하여 자체 코드와 섞이지 않도록 구성
- 법무팀 검토 권장
