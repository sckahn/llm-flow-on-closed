================================================================================
1. Docker Desktop Business - 필수
================================================================================

구매 이유:
- 직원 250명 이상 또는 매출 $10M 이상 기업은 무료 사용 불가
- 라이선스 위반시 법적 조치 가능

가격: $24/user/month (연간 결제시 할인)

구매처: https://www.docker.com/pricing/

포함 기능:
- Docker Desktop 전체 기능
- Docker Scout (보안 스캔)
- Image Access Management
- SSO/SCIM 지원
- 전화/이메일 지원

구매 단위: 개발자 수 기준


================================================================================
2. Neo4j Enterprise - 필수
================================================================================

구매 이유:
- Community Edition은 GPLv3 (소스 공개 의무)
- HA, 클러스터링, RBAC은 Enterprise만 지원
- 500+ 사용자 환경에서 단일 노드는 SPOF 리스크

가격: 별도 견적 (일반적으로 연간 수만~수십만 달러)

구매처: https://neo4j.com/contact-us/

포함 기능:
- Causal Clustering (고가용성)
- Role-Based Access Control
- Hot Backup / Online Backup
- Multi-database
- Property-level security
- 24/7 기술 지원

구매 단위: 코어 수 또는 노드 수 기준


================================================================================
3. MinIO Enterprise (AIStor) - 필수
================================================================================

구매 이유:
- AGPLv3는 네트워크 서비스시 소스 공개 의무
- MinIO는 라이선스 위반에 적극적 소송 (Nutanix, Weka 피소)
- 상용 라이선스로 AGPL 의무 완전 해제

가격:
- Standard: $10/TB/month
- Enterprise: 별도 견적

구매처: https://min.io/pricing

포함 기능:
- AGPL 의무 해제
- Enterprise Console
- Subnet (보안 업데이트)
- 24/7 기술 지원
- SLA 보장
- Audit logging

구매 단위: 스토리지 용량(TB) 기준


================================================================================
4. Redis Enterprise - 필수
================================================================================

구매 이유:
- Redis 7.4+는 SSPL/RSALv2 (오픈소스 아님)
- SSPL은 법적 해석이 모호하고 리스크 존재
- Enterprise 라이선스로 명확한 상용 사용권 확보

가격: 별도 견적

구매처: https://redis.io/pricing/

포함 기능:
- 상용 라이선스 (법적 명확성)
- Active-Active Geo-Distribution
- Redis on Flash
- 99.999% SLA
- 24/7 지원
- Auto Tiering

대안: Valkey (BSD 라이선스, Linux Foundation)
      - 법적으로 안전하나 Redis Inc. 지원 없음
      - 자체 운영 역량 필요

구매 단위: 샤드 수 또는 노드 수 기준


================================================================================
5. Grafana Enterprise - 선택 (권장)
================================================================================

구매 이유:
- OSS는 AGPLv3 (수정시 소스 공개 의무)
- Enterprise 구매시 프로프리어터리 라이선스로 명확
- 고급 기능 (SAML, 리포팅, 감사 로그) 필요시

무료 옵션:
- Enterprise 바이너리 무료 다운로드 가능 (기능 제한)
- 수정 없이 사용시 AGPL 의무 없음

유료 가격: 별도 견적

구매처: https://grafana.com/pricing/

포함 기능:
- 프로프리어터리 라이선스
- Enterprise plugins
- Reporting & PDF export
- Enhanced LDAP/SAML
- Audit logging
- Data source permissions
- 24/7 지원

결정:
- 수정 없이 사용 → 무료 Enterprise 바이너리 OK
- 수정 필요 또는 고급 기능 → 유료 구매


================================================================================
6. Milvus - Zilliz Cloud (선택)
================================================================================

Self-hosted Milvus: Apache 2.0 (무료, 법적 안전)

Zilliz Cloud 구매 이유:
- 관리형 서비스로 운영 부담 감소
- Enterprise 지원 및 SLA

가격: 사용량 기반

구매처: https://zilliz.com/pricing

결정:
- Self-hosted 운영 가능 → Milvus 무료 사용 (Apache 2.0)
- 관리형 원함 → Zilliz Cloud 구매
